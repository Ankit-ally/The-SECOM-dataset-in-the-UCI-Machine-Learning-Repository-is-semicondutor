# The-SECOM-dataset-in-the-UCI-Machine-Learning-Repository-is-semicondutor
## Objective The SECOM dataset presents us with two problems: (i) working with skewed data and (ii) feature selection.  The main focus for this analysis will be the class imbalance issue and the ability to successfully predict fails. Strategies used in fraud/anomaly detection/rare disease diagnosis will be useful here.
